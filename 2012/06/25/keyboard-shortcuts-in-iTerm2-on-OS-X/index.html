keyboard shortcuts in iTerm2 on OS X

I love having my terminal completely on the server, so I can connect from everywhere and just resume with all my windows.

That’s why I always work on tmux and my iTerm just has one tab.

That’s why I remapped e.g. cmd-t to “new tmux tab” and cmd-w to “close tmux tab”.

To achieve this, in tmux’ preferences in keys you need to add global shortcut keys. I’m using ctrl-z as prefix, if you use ctrl-a use 0x01 instead. These are the combinations for ctrl-z (List items are: Key combinations - Hex Codes - Comment):

  • ⌘0 - 0x1a 0x22 - switch to window 0. Other window suffixes (replace 22 by that): 1=31,2=32,3=33,4=34,5=35,6=36,7=37,8=38,9=39
  • ⌘r - 0x1a 0x2c 0x08 0x08 0x08 0x08 0x08 - rename current window (0x08 sends backspace)
  • ⌘t - _0x1a 0x63 - _new “tab” (=new tmux window)
  • ⌘w - 0x04 - sends ctrl-d
  • ⌥⌘← - 0x1a 0x1b 0x5b 0x44 - jump to plane to the left (ctrl-z “ or ctrl-z % to split windows into panes in tmux). Replace 0x44 by 0x41 (up), 0x42 (down), 0x43 (right)
  • ⌘↑ - 0x1a 0x5b - jump to tmux history

Use showkey --ascii on the shell to inspect the hex keys of key strokes.
